## Installation

How to install and run the project

Install node modules, create .env file and enter your parameters
```bash
  npm install 
  copy .env-example .env
```
Compile transcript
```bash
  npx tsc
```
Run migrations
```bash
  node dist/migrations.js
```
Start Project
```bash
  npm start
```
